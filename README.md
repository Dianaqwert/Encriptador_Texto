# Encriptador de Texto

## Descripción del Proyecto

Este proyecto es un encriptador y desencriptador de texto simple creado con HTML, CSS y JavaScript. La aplicación permite a los usuarios ingresar un texto y convertirlo en una versión encriptada siguiendo reglas específicas. Asimismo, permite desencriptar un texto previamente encriptado. Es una herramienta útil para quienes desean ocultar mensajes simples o practicar conceptos de programación web.

## Funcionalidades

- **Encriptación de Texto:** Convierte cualquier palabra compuesta solo de letras minúsculas (sin acentos ni caracteres especiales) en una versión encriptada según las siguientes reglas:
  - 'a' se convierte en 'ai'
  - 'e' se convierte en 'enter'
  - 'i' se convierte en 'imes'
  - 'o' se convierte en 'ober'
  - 'u' se convierte en 'ufat'

- **Desencriptación de Texto:** Convierte una palabra encriptada de vuelta a su forma original.

- **Validación de Entradas:** Solo se permite la encriptación de textos compuestos por letras minúsculas. Si el texto ingresado contiene números, mayúsculas o caracteres especiales, se le notificará al usuario que debe ingresar un dato válido.

- **Interfaz de Usuario Intuitiva:** Una sencilla interfaz gráfica donde los usuarios pueden ingresar texto y escoger si desean encriptarlo o desencriptarlo.

- **Pie de Página Informativo:** El pie de página muestra el nombre del desarrollador del proyecto.

## Cómo pueden usarlo los Usuarios

1. **Uso de la Aplicación:**
   - Ingresa un texto en el campo de entrada (asegúrate de usar solo letras minúsculas sin acentos ni caracteres especiales).
   - Haz clic en el botón "Encriptar" para convertir el texto a su versión encriptada.
   - Haz clic en el botón "Desencriptar" para convertir un texto encriptado de vuelta a su forma original.
   - El resultado será mostrado en pantalla.
